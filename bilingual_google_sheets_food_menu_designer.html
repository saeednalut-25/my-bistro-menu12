How to Publish Your Live Food Menu

Deploy this single-file web app online for free in under 3 minutes.

1 GitHub Pages (Recommended & Free hosting)

  1. Go to github.com and sign up for a free account.
  2. Create a New Repository named my-bistro-menu.
  3. Click "uploading an existing file", then upload this index.html file.
  4. Commit changes, then navigate to the Settings tab of your repository.
  5. Click Pages in the sidebar, set build source to the main branch, and click Save.
  6. Within 1 minute, GitHub will give you a live production link! (e.g. https://username.github.io/my-bistro-menu/).

2 Netlify Drop (Easiest No-Code hosting)

  1. Save this page code to a file named index.html inside a clean folder on your desktop.
  2. Open Netlify Drop in your browser.
  3. Drag and drop your folder containing the file into the upload box on screen.
  4. Your website will deploy instantly with a dynamic, custom link you can share with guests!

How does the data stay synced?

This digital menu utilizes the browser's LocalStorage database engine to store menu adjustments on the manager's device. For professional production setups requiring real-time central updates, you can use the "Export JSON" button to obtain a structural menu file and embed it on standard servers.
